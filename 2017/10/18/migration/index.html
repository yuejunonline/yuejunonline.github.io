家要搬到哪里

发表于 2017-10-18   |   分类于 资源收集

几天来折腾家里的Nas发现80号端口被禁用了，Nas中Win2012的IIS站点非要加上端口才能访问，真是有点郁闷了。Wordpress编辑文章的速度实在是太慢了，可能是我IIS和php搭建环境有问题，后台打开一个文章来编辑需要几十秒，许文章中的图片和附件链接都失效了，需要修复。还是看看搬走吧。可以搬到哪里去呢？其实这个打算早都有了，Wordpress上的文章虽然不多，但是需要一篇一篇搬走，也还真是不太容易，一直下不了决心，可是访问速度不行搞得人都不愿意登陆发帖写作什么的，打开一个页面要很久，修改个帖子要很久，2017年一整年基本没有上去看过。算了，还是狠下心趁最近有空，搬家，搬家吧！
考虑过好几个方案：

新浪SAE平台空间


新浪云应用（Sina App Engine简称SAE）是国内最具影响力的分布式Web应用/业务开发托管、运行平台。SAE是真正意义上具有高可靠、高扩展、免运维的云计算服务。当时处于免费阶段，记得进去装了个低版本的Wrodpress，测试了几天就闲置了，没想到一直在扣云豆，到去年把初始用的1000多云豆都扣完了，成了欠费状态。现在需要充值才能继续使用。国内购买空间和数据库搭建Wordpress成本很高，加上域名的费用，至少160-200元/年，还需要备案。对于我目前的应用来说，没必要花这个钱。

Q友的海外虚拟空间


小马的破解做的很不错的，前几年一直买的Q友小马的海外虚拟空间，每年60元，500M空间、支持Mysql，很经济。去年底一直联系不上这个小马了，加上域名也计划不再续费使用了。个人的海外空间或者主机还是觉得不可靠，钱虽不多，数据无价。还是算了吧。这个方案就不再考虑了。印象中几年前小马还是高中生的时候就买了他的空间，后来得知去了厦大继续读书了。现在不知道什么情况了，这是个挺厉害的IT精英！

申请简书账号搬家

简书是个挺牛的社区，创作社区，任何人均可以在其上进行创作。用户在简书上面可以方便的创作自己的作品，互相交流。简书成为国内优质原创内容输出平台。免费注册账号，写作空间不限制，Warkdown语法编辑，web页面和app都可以写作，很方便也很好用，界面简洁明了，很不错的选择！

对我来说，太高大上了。都是大咖写手搞得原创剧本什么的。我写这几个日记就不要在上面露脸了，还是找个僻静之地留个脚印就可以了。

购买国外Vps主机


Vultr国外优质主机站，有便宜的低价机，最近买了一个512的Debian7x64，用来搭了梯子，近期速度较慢，不知道以后怎么。如果用来架站的话，可以不备案，价格也便宜，就是访问速度不行，本来Wrodpress速度就慢，加上国外延迟都在300以上，用来架站看来也不可行，最主要是便宜主机都是Linux系统，我这二把刀整不明白，所以还是别瞎耽误工夫了。

Hexo + Github Pages


hexo是一款基于Node.js的静态博客框架，Github官方推荐的是Jekyll。对比了下，大多认为hexo比较简单，于是我选择了它。简单明了，同时部署到Github和Coding速度尚可还不要钱，这或许是最佳的方案了。
