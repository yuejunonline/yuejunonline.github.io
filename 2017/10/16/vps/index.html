VPS到手留个记号

发表于 2017-10-16   |   分类于 资源收集

总想看看山的那面有什么，其实山的那面依旧还是山。
SS越来越不好用了，看到google+上有社友留了一个vps搭建的教程，照猫画虎，买了一个，https://www.vultr.com/是一家有名的云服务提供商，在全球共有15个自营数据中心机房，Vultr VPS全部采用KVM架构SSD固态硬盘，不超售，保证性能，支持linux和windows，还可以上传自己的ISO。

购买vps注册

登陆官网https://www.vultr.com/注册好，填写信息。选择了一个Debian 7 x64的vps，512M内存，20Gssd，机房位于New Jersey，价格是2.5$,这是最便宜的一款，比较好的是vultr支持支付宝，试了一下很快很好用瞬间收钱。充值10$，每月扣2.5$，这个价钱想一想其实也不算便宜。

安装Amh面板

安装了ss服务端以后，内存大约只使用了4分之一，别浪费，搞个虚拟主机什么的玩一玩。继续照猫画虎！安装amh面板，不太了解这个东西，其实就是代替命令行模式的web套件，对于我这样的新手来说非常需要，直达官网，按照用户手册执行安装。

AMH 4.2 为独立的一套LNMP/Nginx虚拟主机面板，安装请使用纯净系统。
编译安装方式 (安装时间15至25分钟)
1) 使用 root 账号登录 Linux 服务器。
2) 执行 amh 安装脚本。
wget http://amh.sh/file/AMH/4.2/amh.sh && chmod 775 amh.sh && ./amh.sh 2>&1 | tee amh.log
3) 根据提示输入选择1~3选项。1为安装amh，2为卸载amh，3为退出不做操作。
输入1回车，接着输入MySQL与AMH密码即进入安装流程，安装过程大约需10至20分钟(以服务器性能为准)，最后如看到安装成功提示，说明系统已安装完成。
成功安装后有必要请删除日志文件:amh.log，如安装失败需协助安装请把错误日志反馈我们。
访问http://ip:8888 即可进入AMH web端管理，默认账号为admin。
安装小提示
1、安装amh成功后无法访问面板情况，请您检查主机商是否有安全组/防火墙相关限制，请开放端口，如开放面板8888端口。
2、安装amh过程中如果服务器解析不了amh.sh域名的情况，
请先更换服务器的DNS再安装AMH。执行以下命令完成更换DNS解析：
echo -e “options timeout:1 attempts:1 rotate\nnameserver 8.8.8.8\nnameserver 114.114.114.114” >/etc/resolv.conf;
